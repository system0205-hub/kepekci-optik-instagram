# Sablon: Bilgi Reels

## Format
- **Tip:** Reels (dikey video)
- **Boyut:** 1080x1920 (9:16)
- **Sure:** 15-30 saniye
- **Ton:** Samimi + dinamik

## Sahne Yapisi

### Hook (0-3 saniye) — EN ONEMLI KISIM
```
[SORU veya SASIRTICI BILGI]
Metin overlay: Buyuk, okunabilir font
Ornek hook'lar:
- "Biliyor muydunuz?"
- "Bunu bilmeden gozluk almayin!"
- "Cocugunuzun gozleri tehlikede olabilir!"
- "%90'i bu hatayi yapiyor!"
```

### Icerik (3-20 saniye)
```
[BILGI GORSELLERI + METIN OVERLAY]
- Her 3-5 saniyede yeni gorsel/sahne
- Kisa, okunabilir metin overlay'ler
- Goz alici gecisler
```

### CTA (son 3-5 saniye)
```
[MAGAZAYA DAVET]
- "Detayli bilgi icin magazamiza gelin!"
- Logo + iletisim bilgisi
- "Takip edin, daha fazla bilgi icin!"
```

### Logo (TUM VIDEO BOYUNCA)
```
Sag alt kose, %8-10 boyut, %60-70 saydam watermark
```

## Aciklama Sablonu
```
[Hook — video basligini tekrarla]

[1-2 cumle ek bilgi]

[CTA]

Yeni hastane yolu uzeri, Kilis
0506 905 55 70

#[genel] #[yerel] #[konuozel]
```

## 4 Reels Formulu

### Formula 1: "Biliyor muydun?"
```
0-3sn: "Biliyor muydunuz? [sok bilgi]"
3-12sn: Aciklama + gorsel
12-15sn: CTA + logo
```

### Formula 2: "Once & Sonra"
```
0-3sn: "Once vs Sonra"
3-8sn: Once durumu (sorunlu)
8-13sn: Sonra durumu (cozum)
13-15sn: CTA + logo
```

### Formula 3: "3 Sey"
```
0-3sn: "[Konu] hakkinda 3 onemli bilgi"
3-7sn: 1. madde
7-11sn: 2. madde
11-15sn: 3. madde + CTA
```

### Formula 4: "POV"
```
0-3sn: "POV: [durum]"
3-12sn: Deneyim gorselleri
12-15sn: Logo + CTA
```

## Ornek: "Mavi Isik Gozlerinize Ne Yapiyor?"

**Hook (0-3sn):** "Ekran basinda gunluk 7+ saat geciriyorsaniz bunu bilin!"
**Icerik (3-12sn):**
- Sahne 1: Bilgisayar basinda yorgun gozler
- Sahne 2: Mavi isik dalga boyu gorseli
- Sahne 3: Mavi isik filtreli cam gosterimi
**CTA (12-15sn):** "Mavi isik filtreli camlar magazamizda!"
**Logo:** Surekli sag alt
