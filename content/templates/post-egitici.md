# Sablon: Egitici Post

## Format
- **Tip:** Tek gorsel (dikey)
- **Boyut:** 1080x1350 (4:5)
- **Ton:** Samimi + bilgilendirici

## Gorsel Yapisi
```
┌─────────────────────┐
│   [BASLIK SORUSU]   │ ← Turkuaz arka plan
│                     │
│   [ANA GORSEL/IKON] │ ← Konuya uygun gorsel
│                     │
│   [2-3 SATIR BILGI] │ ← Koyu gri metin
│                     │
│   [KEPEKCI OPTIK]   │ ← Logo + slogan
│   Gozunuz Bizde     │
│       Olsun         │
└─────────────────────┘
```

## Aciklama Sablonu
```
[Konu sorusu — hook]

[2-3 cumle bilgi — samimi dil]

[CTA: Magazaya davet]

Detayli bilgi icin magazamiza bekleriz
Yeni hastane yolu uzeri, Kilis
0506 905 55 70

#[genel] #[yerel] #[konuozel]
```

## Ornek: Progresif Cam
**Gorsel baslik:** "Progresif Cam Nedir?"
**Aciklama:**
```
Tek camda hem uzagi hem yakini gormek mumkun mu?

Progresif camlar sayesinde artik gozluk degistirme derdiniz bitiyor! Uzak, orta ve yakin mesafe — hepsi tek bir camda. Bifokal cizgisi yok, modern gorunum var.

40 yas ustu gorme problemleriniz icin en konforlu cozum!

Detayli bilgi icin magazamiza bekleriz
Yeni hastane yolu uzeri, Kilis
0506 905 55 70

#progresifcam #kilisoptik #gozluk
```

## Canva Tasarim Notu
- Arka plan: Beyaz veya acik gri
- Baslik: Turkuaz (#2EC4B6), Montserrat Bold
- Metin: Koyu gri (#3A3A3A), Open Sans
- Logo: Alt kisim, tam opak
- Ikon/gorsel: Konuyla ilgili flat design ikon
