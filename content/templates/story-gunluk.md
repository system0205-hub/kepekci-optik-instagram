# Sablon: Story (Gunluk/Kampanya)

## Format
- **Tip:** Story (dikey)
- **Boyut:** 1080x1920 (9:16)
- **Sure:** 15 saniye (gorsel) veya 15sn (video)
- **Ton:** Sicak + samimi veya Dinamik (kampanya)

## Gorsel Yapisi
```
┌─────────────────────┐
│ [KEPEKCI OPTIK]     │ ← Logo (ust)
│                     │
│                     │
│   [ANA GORSEL]      │ ← Foto veya video
│                     │
│                     │
│   [METIN OVERLAY]   │ ← Kisa, okunabilir
│                     │
│   [STICKER/ANKET]   │ ← Interaktif eleman
│                     │
│   [CTA / LINK]      │ ← "Yukari kaydir" veya link
└─────────────────────┘
```

## Story Tipleri

### Tip 1: Gunluk Magaza
- Magaza icinden foto/video
- "Bugun magazamizda..." tonuyla
- Yeni urunler, vitrin, ekip

### Tip 2: Kampanya Duyuru
- Bold renkler, indirim orani
- Aciliyet hissi ("Bugun son gun!")
- Link sticker'i (web sitesi)

### Tip 3: Anket/Soru (Engagement)
- "Hangi cerceve daha cok yakisti?"
- "Gunes gozlugu sezonuna hazir misiniz?"
- Anket sticker'i kullan

### Tip 4: Bilgi Capsule
- 1 gorsel + 1 bilgi maddesi
- Hizli tuketilecek icerik
- Seriye bagla (3-5 story art arda)

## Pencil Tasarim Notu
- Metin overlay: ekranin alt %40'inda (ust kisim profil vs tarafindan kapanabilir)
- Font: Buyuk, okunabilir (min 24px)
- Arka plan: Foto uzerine yari saydam renk katmani
- Logo: Ust kisim veya watermark
- Interaktif: Anket, soru kutusu, emoji slider sticker alani birak
