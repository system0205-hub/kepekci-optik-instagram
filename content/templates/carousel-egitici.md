# Sablon: Egitici Carousel (Swipe)

## Format
- **Tip:** Carousel (4-5 slide)
- **Boyut:** 1080x1350 (4:5) — tum slide'lar ayni boyut
- **Ton:** Samimi + bilgilendirici

## Slide Yapisi

### Slide 1: Kapak (Hook)
```
┌─────────────────────┐
│                     │
│   [HOOK SORUSU]     │ ← Buyuk, dikkat cekici
│   veya              │
│   [SASIRTICI BILGI] │
│                     │
│   ← Kaydir          │ ← Swipe gostergesi
│   [KEPEKCI OPTIK]   │ ← Logo
└─────────────────────┘
```

### Slide 2-3-4: Bilgi Slide'lari
```
┌─────────────────────┐
│   [MADDE NUMARASI]  │ ← 1, 2, 3 (turkuaz daire)
│                     │
│   [GORSEL/IKON]     │ ← Konuyla ilgili
│                     │
│   [BASLIK]          │ ← Montserrat Bold
│   [ACIKLAMA]        │ ← Open Sans, 2-3 satir
│                     │
│   [KEPEKCI OPTIK]   │ ← Logo (kucuk, sag alt)
└─────────────────────┘
```

### Slide 5: Kapani (CTA)
```
┌─────────────────────┐
│                     │
│   [OZET/SONUC]      │ ← Ana mesaj tekrar
│                     │
│   [CTA]             │ ← "Magazamiza gelin!"
│                     │
│   [ILETISIM]        │ ← Adres + telefon
│                     │
│   [KEPEKCI OPTIK]   │ ← Logo + slogan (buyuk)
│   Gozunuz Bizde     │
│       Olsun         │
└─────────────────────┘
```

## Aciklama Sablonu
```
[Hook sorusu veya baslik]

[Kaydir ve ogren! / Detaylari gormek icin kaydir →]

[Ozet bilgi — 1-2 cumle]

Kaydedin, paylasim!

Yeni hastane yolu uzeri, Kilis
0506 905 55 70

#[genel] #[yerel] #[konuozel]
```

## Ornek: "Progresif Cam Nedir?"

**Slide 1:** "Progresif Cam Nedir? Tek Camda 3 Mesafe!" + swipe gostergesi
**Slide 2:** "1. Uzak Gorus" — TV izleme, araba kullanma icin ust bolge
**Slide 3:** "2. Orta Mesafe" — Bilgisayar, raf icin orta bolge
**Slide 4:** "3. Yakin Gorus" — Okuma, telefon icin alt bolge
**Slide 5:** "Gozluk degistirme derdiniz bitiyor!" + CTA + iletisim

**Aciklama:**
```
Progresif cam nedir biliyor musunuz?

Tek camda hem uzagi, hem ortayi, hem yakini gorun! Kaydir ve ogren →

40 yas ustu gorme problemleriniz icin en konforlu cozum. Bifokal cizgisi yok, modern gorunum var.

Kaydedin, ihtiyaci olan arkadaslarinizla paylasin!

Yeni hastane yolu uzeri, Kilis
0506 905 55 70

#progresifcam #kilisoptik #gozluk
```

## Canva Tasarim Notu
- Tum slide'lar AYNI gorsel dil (renk, font, stil)
- Arka plan: Beyaz veya acik gri (temiz)
- Numara daireleri: Turkuaz (#2EC4B6) daire icinde beyaz rakam
- Basliklar: Koyu gri, Montserrat Bold
- Icerik: Open Sans Regular
- Logo: Her slide'da (ilk ve son buyuk, ara slide'larda kucuk)
- Swipe gostergesi: Ilk slide'da ok veya "← Kaydir" yazisi
