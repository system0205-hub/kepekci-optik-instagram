# Sablon: Urun Tanitim Post

## Format
- **Tip:** Tek gorsel (dikey veya kare)
- **Boyut:** 1080x1350 (4:5) veya 1080x1080 (1:1)
- **Ton:** Profesyonel + cekici

## Gorsel Yapisi
```
┌─────────────────────┐
│   [MARKA LOGOSU]    │ ← Markanin kendi logosu
│                     │
│   [URUN GORSELI]    │ ← Buyuk, net urun foto
│                     │
│   [MODEL ADI]       │ ← Montserrat Bold
│   [OZELLIKLER]      │ ← 2-3 madde
│                     │
│   [KEPEKCI OPTIK]   │ ← Magaza logosu
└─────────────────────┘
```

## Aciklama Sablonu
```
[Marka] [Model Adi] [Emoji]

[1-2 cumle urun ozelligi — profesyonel dil]

[Ozellik listesi (opsiyonel)]
```

## Canva Tasarim Notu
- Arka plan: Beyaz veya marka rengine uygun gradient
- Urun foto: Merkez, buyuk, net
- Marka logosu: Ust kisim (kucuk)
- Magaza logosu: Alt kisim
- Font: Montserrat Bold (model adi), Open Sans (ozellikler)
