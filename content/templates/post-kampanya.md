# Sablon: Kampanya/Indirim Post

## Format
- **Tip:** Tek gorsel (dikey)
- **Boyut:** 1080x1350 (4:5)
- **Ton:** Dinamik + aciliyet

## Gorsel Yapisi
```
┌─────────────────────┐
│   [KAMPANYA BASLIGI] │ ← Bold, buyuk, dikkat cekici
│                      │
│   [INDIRIM ORANI]    │ ← %30, %40, %50 — KIRMIZI/ALTIN
│                      │
│   [URUN GORSELI]     │ ← Indirimli urun/kategori
│                      │
│   [SURE/TARIH]       │ ← "Son 3 gun!" "Bu hafta sonu!"
│                      │
│   [KEPEKCI OPTIK]    │ ← Logo
└──────────────────────┘
```

## Aciklama Sablonu
```
[Aciliyet hook — buyuk harf veya emoji]

[Kampanya detayi — ne, ne kadar, ne zamana kadar]

[Urun/kategori detayi]

Firsati kacirmayin!

Yeni hastane yolu uzeri, Kilis
0506 905 55 70

#[kampanya] #[yerel] #[genel]
```

## Ornek: Yaz Indirimi
**Gorsel:** Gunes gozlukleri kolaji + "%40'a varan indirim" metni
**Aciklama:**
```
YAZ INDIRIMI BASLADI!

Gunes gozluklerinde %40'a varan indirim firsatini kacirmayin!

Ray-Ban, Prada, Miu Miu, Oakley ve daha bircok markada gecerli.

Son gun: 30 Haziran

Yeni hastane yolu uzeri, Kilis
0506 905 55 70

#indirim #kilis #gunesgozlugu
```

## Canva Tasarim Notu
- Arka plan: Canli renkler (altin gradient, turkuaz, veya kirmizi tonlar)
- Indirim orani: Buyuk font, kirmizi (#E63946) veya altin (#F4A261)
- Sure/tarih: Vurgulu, aciliyet hissi
- Logo: Sag alt
- Dikkat: Cok kalabalik yapma, indirim orani ONE CIKMALI
