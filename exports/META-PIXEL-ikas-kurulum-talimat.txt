================================================================================
GÖREV 1 — Meta'da Pixel Oluştur (henüz yoksa)
================================================================================

1. https://business.facebook.com adresine git
2. Sol menüden "Events Manager" (Etkinlik Yöneticisi) seç
3. "Connect Data Sources" (Veri Kaynağı Bağla) → "Web" seç
4. "Connect" (Bağla) butonuna tıkla
5. "Create a new dataset" (Yeni veri kümesi oluştur) seç
6. İsim: Kepekçi Optik Pixel
7. Website URL: https://kepekcioptik.com
8. "Create" (Oluştur) tıkla
9. **PIXEL ID'SINI KOPYALA** — 15-16 haneli sayı (örn: 1234567890123456)

Bu ID'yi KAYDET, ikas'a girerken kullanacağız.

================================================================================
GÖREV 2 — ikas'ta Meta Pixel Aktifleştir
================================================================================

ikas'ın Meta entegrasyonu var. Manuel kod eklemene gerek yok.

1. ikas Admin Panel'e git
2. Sol menüden "Pazarlama" (Marketing) → "Facebook ve Instagram" seç
   VEYA
   "Ayarlar" → "Entegrasyonlar" → "Meta" / "Facebook" bölümünü bul
3. "Pixel Bağla" veya "Meta Pixel Entegrasyonu" butonunu tıkla
4. Adım adım gelen ekranda:
   - Facebook hesabınla giriş yap
   - Kepekçi Optik Business Manager'ı seç
   - Kepekçi Optik Pixel'i seç (az önce oluşturduğun)
5. "Bağla" ve "Onayla" tıkla
6. Kaydet

Eğer ikas'ın hazır Meta entegrasyonunu bulamazsan:

ALTERNATIF — Google Tag Manager ile Pixel Ekle
------------------------------------------------
GTM zaten kurulu (GTM-KSXLL2NW). Pixel'i GTM üzerinden de ekleyebiliriz:

1. https://tagmanager.google.com → Kepekci Optik workspace
2. "Etiketler" → "Yeni"
3. Tag adı: Meta Pixel - Kepekçi Optik
4. Tag Configuration → "Facebook Pixel" template ara
5. Yoksa "Custom HTML" seç ve şu kodu yapıştır:

================================================================================
GÖREV 3 — Meta Pixel Doğrulama
================================================================================

Pixel'in çalıştığını kontrol et:

1. Chrome'a "Meta Pixel Helper" extension'ı kur:
   https://chromewebstore.google.com/detail/meta-pixel-helper/fdgfkebogiimcoedlicjlajpkdmockpc

2. kepekcioptik.com adresini aç
3. Sağ üstte Meta Pixel Helper ikonu yeşil "1 pixel found" göstermeli
4. Tıkla → "PageView" event'i gelmeli

Eğer yeşil değilse:
- 5 dakika bekle (cache yenilenmesi)
- Tekrar dene
- Hala olmuyorsa kurulum hatası var, tekrar kontrol et

================================================================================
GÖREV 4 — Conversion API (Önerilir ama opsiyonel)
================================================================================

iOS 14.5 sonrası iPhone kullanıcıları Pixel takibini engelleyebiliyor.
Conversion API (CAPI) bu soruna çözüm — satın almayı ikas'tan direkt Meta'ya
gönderiyor. ikas'ın bu entegrasyonu varsa hemen aç:

1. ikas Admin → Pazarlama → Facebook ve Instagram
2. "Conversion API" veya "CAPI" bölümünü bul
3. Aktif et
4. Access Token iste → Meta Business Manager'dan "System User Token" üret:
   - business.facebook.com → Settings → Users → System Users
   - "Add" → "Name: ikas CAPI, Role: Admin"
   - "Generate New Token" → Asset: Pixel
   - Token'ı kopyala → ikas'a yapıştır
